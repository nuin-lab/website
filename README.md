## Overview

Astro with Content CollectionsによるSSGで構築されているウェブサイトのソース

## Local

### Requirements

- Node.js v24以降

### Developing

以下のいずれかで開発サーバーを起動する。

```sh
pnpm run dev

# サーバーを起動してデフォルトブラウザで開く
pnpm run dev -- --open
```

### Building

リリースver.を作成するには以下を実行する。

```sh
pnpm run build

# プレビューまで表示したい場合は代わりにこちらを実行
pnpm run preview
```
